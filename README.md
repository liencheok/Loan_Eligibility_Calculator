This project was developed as part of the **Business Analytics** course.  
It is a Python-based console application that calculates loan repayment details and evaluates loan eligibility using the Debt Service Ratio (DSR).

## Project Overview
The program allows users to:
- Calculate monthly loan instalments
- Compute total amount payable over the loan term
- Evaluate Debt Service Ratio (DSR)
- Determine loan eligibility based on DSR threshold
- Store and review multiple loan calculations during runtime

This project demonstrates the application of business rules and financial concepts using Python.

## Key Features
- Loan instalment calculation using compound interest formula
- DSR calculation based on monthly income and commitments
- Eligibility decision logic (Approved / Denied)
- Input validation for numerical and positive values
- Menu-driven console interface
- Stores multiple loan calculations in memory

## Technologies Used
- Python
- Basic financial mathematics
- Console-based user interface
